### Installation
```python
pip3 install justgood
pip3 install thrift
pip3 install json
pip3 install pytz
pip3 install datetime
pip3 install requests
```

### How to run ?
``` python
$ git clone https://github.com/RendyTR/LINE-SelfBOT
$ cd LINE-Selfbot
$ python3 client.py
```
